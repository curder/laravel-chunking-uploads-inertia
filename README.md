# Laravel chunking uploads by inertia

通过分块、可恢复上传，轻松处理 Inertia/Vue 应用程序中的大文件上传。

通过介绍客户端和服务器的整个上传过程，显示进度条，然后添加暂停、恢复和取消上传的功能。

将能够在应用程序中的任何位置处理巨大的文件上传。
